# Compilador y Ejercicios en Go
Ejercicios y desarrollo del proyecto de la clase TC3002B Desarrollo de aplicaciones avanzadas de ciencias computacionales

Este repositorio se usara para practicar usar el lenguaje Go, y acostumbrar a usar mas Github en general.

## El Compilador **_Patito_**
El compilador Patito es un proyecto que se esta desarollando en el lenguaje Go, usando como guia _Writing an Interpreter in Go_ por Thorsten Ball y la herramienta goyacc para su implementación.
